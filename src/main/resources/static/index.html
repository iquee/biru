README

by Luiz Henrique K Taira

This is a simple spring-boot application to create and get a single PDV(Ponto de Venda). There is also a service to find the nearest PDV from a specific location. technologies:

  • - Spring boot as MVC framework: v 2.1.5.RELEASE
    • starter-data-jpa
    • starter-data-rest
    • starter-data-web
    • starter-data-security
    • starter-data-test
    • spring-data-mongodb
  • - MongoDB as NoSQL databases
  • - Swagger for API documentation
  • - JUnit to unity tests
  • - Maven as project dependency management

  • - Travis-CI as platform to run the tests
  • - Github as source repository
  • - Docker to run in a container
